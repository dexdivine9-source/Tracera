# Tracera

**Real-time Solana pulse across Africa.**

Tracera is the real-time Solana Africa ecosystem intelligence dashboard. Think of it as DefiLlama + Dune + Crunchbase specifically tailored for the Solana ecosystem in Africa.

## Product Vision
To provide unparalleled visibility into the adoption, growth, and impact of Solana across the African continent, empowering builders, investors, and researchers with real-time, actionable intelligence.

## Tech Stack (2026)
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Database & Auth:** Supabase
- **Blockchain Data:** Helius RPC
- **DeFi Data:** DefiLlama API

## 6-Week Roadmap
- **Week 1:** Foundation & Scaffold (Next.js 15, Tailwind, Supabase setup, Landing Page)
- **Week 2:** Data Ingestion Pipeline (Helius & DefiLlama integrations)
- **Week 3:** Dashboard UI & Core Metrics (Active wallets, TVL, Volume)
- **Week 4:** Project Directory & Crunchbase-style profiles
- **Week 5:** Real-time Updates & WebSockets
- **Week 6:** Polish, Mobile Optimization, & Launch

## Getting Started
1. Clone the repository
2. Copy `.env.example` to `.env.local` and fill in your credentials
3. Run `npm install`
4. Run `npm run dev`

**Note: Open http://localhost:3000 to see the landing page, and http://localhost:3000/dashboard to see the full dashboard.**
